# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus feedback on ideas and code, not people.
- Help newcomers get unstuck.

## Unacceptable Behavior

- Harassment, discrimination, or hateful language.
- Personal attacks or sustained hostility.
- Sharing private information without consent.
- Spam, trolling, or intentionally disruptive behavior.

## Enforcement

Project maintainers may remove comments, lock threads, or restrict access for behavior that violates this code.

## Reporting

If you experience or witness unacceptable behavior, open a private security advisory in the repository security tab.
